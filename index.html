How It Works

A focused workflow instead of an imaginary magic keyboard.

Liaison Keyboard is live today as a standalone web and app experience. The current product is intentionally simple: paste the incoming message, select the tone you want, and copy the reply that fits best.

1. Paste the incoming message Use the composer to drop in the message you received and set the context.
2. Choose the persona Switch between available tones and unlock Pro strategists when the situation needs more nuance.
3. Generate and copy the best fit Review three reply styles, copy one, or copy them all for fast iteration.
